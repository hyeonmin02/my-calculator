🧮 계산기 과제
---
🧱 STEP 1. 클래스 없이 기본 연산 계산기 구현
---
목표
- **클래스를 사용하지 않고, 기본적인 사칙연산이 가능한 콘솔 계산기를 구현**
---
📌 구현 요구사항
---
양의 정수(0 포함)를 입력받기

사칙연산 기호 ➕, ➖, ✖️, ➗ 를 입력받기

위에서 입력받은 양의 정수 2개와 사칙연산 기호를 사용하여 연산을 진행한 후 결과값을 출력하기

반복문을 사용하되, 반복의 종료를 알려주는 “exit” 문자열을 입력하기 전까지 무한으로 계산을 진행할 수 있도록 소스 코드를 수정하기

---
🧱 STEP 2. 클래스 기반 계산기 구현
---
목표
- **Calculator 클래스 생성해서 연산 책임을 App(main)에서 분리**

- **Getter와 Setter를 이용해 결과 저장 및 삭제 구현**

- **STEP1에서의 기존 기능 유지**

📌 구현 요구사항
---
사칙연산을 수행한 뒤 결과를 반환하는 메서드 구현

연산 결과를 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스 생성

양의 정수 2개(0 포함)와 연산 기호를 매개변수로 받아
사칙연산(➕, ➖, ✖️, ➗)을 수행하고 결과를 반환하는 메서드 구현

STEP 1에서 구현한 App 클래스의 main 메서드에서
Calculator 클래스를 활용하도록 구조 수정

App 클래스에서 Calculator의 연산 결과 컬렉션 필드에
직접 접근하지 못하도록 캡슐화 적용 (Getter 방식 사용)
